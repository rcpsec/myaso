Overview

Duck Punch of TokyoCabinet Hash database that uses JSON over Yajl to serialize Ruby objects. Serialization necessary since TC expects values as strings or integers so won’t handle other data types. Also provides a consistent, simpler open method.

This implementation is based on article: blog.henriquez.net/2009/07/tokyo-cabinet-and-me.html

def open(path_to_db)
  unless self.original_open(path_to_db, HDB::OWRITER | HDB::OCREAT)
    raise "#{path_to_db} error: #{self.errmsg(self.ecode)}"
  end
end
